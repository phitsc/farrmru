===============================================================================

FARR MostRecentlyUsed plugin - 0.0.0.1 - 27. June 2008

===============================================================================

Description
============
Shows the contents of 'My Recent Documents' in FARR.

Additionally, can show most recently used files of applications storing their most recently used files in the registry.


Installation
============
Copy all files into the FarrMostRecentlyUsed plugin directory (usually C:\Program Files\FindAndRunRobot\Plugins\FarrMostRecentlyUsed) and restart FARR.


Options
============
- IgnoreDirectories
